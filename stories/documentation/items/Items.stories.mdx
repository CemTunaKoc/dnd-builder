# Items

Items are Object and contains some properties which is used by Components.

Every Items Should contain 4 properties which are

Property Name           | Type
---                     | ---
`Component`             | React Component
`details`               | Object
`itemType`              | String
`settings`              | Object
`defaultItem`(optional) | Object


## _Here is the basic Item Object example_
  ```javascript
  const Image = {
    Component: ImageElement,
    details,
    itemType: 'image',
    settings,
  };
  ```

- `Component` is expected to be a React component.
- `details` are default properties to be set to item when added for the first time. This properties merge to item which you get from onItemadd function parameter.
- `itemType` is unique and used for to coupling items to render. *_`layout` & `page` are reserved types so do not use them for items_*
- `settings` is an array of config objects that will be used in rendering the settings panel of the item.
